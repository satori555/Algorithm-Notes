# 47 全排列 II

给定一个可以包含重复数字的序列，返回所有不重复的全排列。

##### 示例:

```
输入: [1,1,2]
输出:
[
  [1,1,2],
  [1,2,1],
  [2,1,1]
]
```

##### 解法：

可以用和上一题完全一样的解法，在追加时判断列表中是否已经有这个结果：

```python
class Solution:
    def permuteUnique(self, nums: List[int]) -> List[List[int]]:
        n = len(nums)
        res = []
        def backtrack(start):
            if start == n and nums[:] not in res:
                res.append(nums[:])
            for i in range(start, n):
                nums[i], nums[start] = nums[start], nums[i]
                backtrack(start+1)
                nums[i], nums[start] = nums[start], nums[i]
        backtrack(0)
        return res
```
